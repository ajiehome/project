### 项目简介

原有的mx4快速拨号功能将作为 【浏览器Home页】概念,以新的表现形式展⽰示给⽤用户.
通过个性化的定制需求增加⽤用户的使⽤用率,达到增强产品的变现能⼒力。

### 功能介绍

* 1、个人信息模块。
* 2、搜索引擎，接入常用的搜索引擎。
* 3、我的站点管理。
* 4、背景图片管理：动态背景、自定义上传图片，支持GIF等。

### 技术方案

MX4的技术方案，作为浏览器的内置页，随用户安装到本地。
优点：与客户端绑定，媲美原生应用的体验；
缺点：更新方式不灵活，需要客户端推更。

MX5新的快速拨号，采用嵌入远程web页面的方式。
优点：作为web站点，部署灵活，不占用安装包大小。
缺点：恶心的版本判断以及不支持离线消息推送。
